# Secure Streaming Network using VLANs – Cisco Packet Tracer

Proyecto académico enfocado en el diseño e implementación de una red virtual segura
para transmisión de streaming.

## Tecnologías
- Cisco Packet Tracer
- Cisco IOS
- VLANs
- ACLs
- Routing Inter-VLAN
- SSH / HTTP

## Descripción
La red está segmentada en VLANs para administración, clientes e invitados,
permitiendo aislamiento de tráfico, control de acceso y transmisión segura de contenido multimedia.

## Funcionalidades
- Segmentación lógica mediante VLANs
- Enrutamiento Router-on-a-Stick
- Control de tráfico con ACLs
- Acceso administrativo seguro por SSH
- Simulación de servidor de streaming HTTP
